# JavaScript Date Methods (Assignment #31-34)

## Topics Covered
- Getting current date and time using `new Date()`
- Extracting day, month, year, hours using date methods
- Creating custom dates
- Modifying dates using `setFullYear()`, `setHours()`
- Calculating time differences between dates
